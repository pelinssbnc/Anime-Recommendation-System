# 🎌 Anime Öneri Sistemi

Kullanıcı puanlama verilerine dayalı olarak kişiselleştirilmiş
anime önerileri sunan makine öğrenmesi projesi.

---

## 📌 Genel Bakış

Bu projede kullanıcıların izledikleri ve puanladıkları animeler analiz edilerek
benzer zevklere sahip kullanıcılara uygun içerikler önerilmektedir.
İşbirlikçi ve içerik tabanlı filtreleme yöntemleri karşılaştırılmaktadır.

Temel hedefler:
- Kullanıcı-öğe etkileşim matrisini oluşturmak
- İşbirlikçi filtreleme modeli geliştirmek
- RMSE ve Precision@K metrikleriyle değerlendirmek
